## 4. BetterGI `Assets/Model/LICENSE` Scope Analysis

The `Assets/Model/LICENSE` GPLv3 preamble states:

> "All models trained by the BetterGI organization within the current directory are distributed under the GNU General Public License version 3 (GPLv3)."

Key observations:
1. **"models trained by"** — This specifically refers to models that BetterGI trained/created, not all files in the directory
2. **"full-chain" copyleft** — Any code interacting with or influenced by these models must be GPLv3 — but this applies to the code, not to third-party model weights
3. **PaddleOCR models were not "trained by BetterGI"** — they were converted from PaddleOCR inference models using Paddle2ONNX
4. **The LICENSE is redundant for code** — the repo root LICENSE already covers all code under GPLv3

## 5. Redistribution Architecture Implications

### 5.1 BetterGI-mac is GPLv3

Since BetterGI-mac is already GPLv3, GPL-3.0 model weights (Yap) are compatible. No additional license conflict.

### 5.2 PaddleOCR Apache 2.0 models in GPLv3 distribution

Apache 2.0 code/models can be included in a GPLv3 distribution. The Apache 2.0 attribution requirements (Section 4 of Apache 2.0) must be satisfied:
- Include a copy of the Apache 2.0 license
- Preserve copyright notices
- Include any NOTICE file if present

### 5.3 Preheat PNGs — the remaining risk

PNG files have no license evidence. They cannot be included in a redistribution source-lock without verification of their copyright status.
